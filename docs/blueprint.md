# **App Name**: تواصل (Tawasul)

## Core Features:

- مصادقة المستخدم (User Authentication): تسجيل دخول وإنشاء حساب آمن للمستخدمين الجدد.
- إنشاء مشاركات (Create Posts): يمكن للمستخدمين كتابة ونشر منشورات نصية قصيرة على حساباتهم.
- خلاصة المشاركات (Home Feed): عرض منشورات المستخدمين الذين يتابعونهم في تدفق سهل التصفح.
- الملفات الشخصية للمستخدمين (User Profiles): صفحة مخصصة لكل مستخدم تعرض معلوماته ومنشوراته.
- متابعة/إلغاء المتابعة (Follow/Unfollow): قدرة المستخدمين على متابعة حسابات أخرى لتخصيص خلاصة المشاركات الخاصة بهم.
- أداة تحسين المنشورات بالذكاء الاصطناعي (AI Post Enhancer Tool): أداة مدعومة بالذكاء الاصطناعي تقترح علامات التجزئة (hashtags) وجملة واحدة (جمده) لتعزيز المنشورات. عند التفاعل معها، ستعرض الأداة رسالة للمستخدمين تفيد بأن ميزات الإعجاب والتعليق وإعادة النشر للمنشورات ستكون متاحة قريباً.
- الإعجاب بالمشاركات (Like Posts): التفاعل مع المنشورات بإبداء الإعجاب لتعزيز التفاعل بين المستخدمين.

## Style Guidelines:

- Primary color: A calm, professional blue-grey (#506899), reflecting reliability and thoughtful communication.
- Background color: A very light, almost off-white blue-grey (#F3F5FA) for a clean, expansive feel, supporting the app's professional yet simple aesthetic.
- Accent color: A fresh, vibrant aqua (#42C1DD) to draw attention to interactive elements and highlight new content, symbolizing connection and clarity.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean lines, and excellent readability across various text sizes, ensuring a professional and accessible experience.
- Use minimalist and universally understood icons to maintain a clean interface and facilitate intuitive navigation for Arabic-speaking users.
- Implement a clean, content-first layout with generous white space to enhance readability and ensure a smooth user experience across all devices, prioritizing Arabic text flow.
- Subtle and functional animations that provide positive feedback on user actions without distracting from the main content, contributing to a polished and responsive feel.
